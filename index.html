🔄

อัปเดตตัวเองได้

มีเวอร์ชันใหม่เมื่อไหร่แจ้งทันที กดปุ่มเดียว ติดตั้งแล้วเปิดใหม่ให้อัตโนมัติ

🔒

ข้อมูลไม่ออกจากเครื่อง

ใช้ token ของ Claude Code ในเครื่องคุณเอง คุยตรงกับ Anthropic เท่านั้น ไม่มี server คนกลาง

ติดตั้งใน 3 ขั้น

1

ดาวน์โหลดแล้วแตก zip จะได้ ClaudeUsageWidget.app — ลากไปไว้ใน Applications

2

ดับเบิ้ลคลิกเปิดแอปก่อน เมื่อ macOS ขึ้นเตือนให้กด Done จากนั้นไปที่ System Settings → Privacy & Security → กด Open Anyway (ปุ่มนี้จะแสดงหลังจากลองเปิดแอปแล้วเท่านั้น)
หรือใน Terminal: xattr -d com.apple.quarantine /Applications/ClaudeUsageWidget.app
(แอปยังไม่ได้ notarize กับ Apple เลยต้องยืนยันเองหนึ่งครั้ง — ทำครั้งเดียวพอ)

3

ล็อกอิน Claude Code ไว้ในเครื่อง แล้วดูโควตาได้จาก menu bar เลย
